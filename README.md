# Простой сканер портов на Python 🚀

Скрипт для автоматизированного аудита сетевой безопасности, написанный на Python внутри среды Kali Linux (WSL). Предназначен для быстрого сканирования хостов и обнаружения открытых сетевых портов.

## 🛠 Какие технологии использовались:
* **Python 3**
* Встроенная библиотека **socket** (для работы с сетевыми подключениями на низком уровне)
* Операционная система **Kali Linux**

## 🎯 Как это работает:
Скрипт поочередно создает TCP-сокеты и пытается подключиться к заданному списку популярных портов (21, 22, 80, 443, 8080) с таймаутом в 2 секунды. Если порт отвечает — утилита сообщает, что сервис активен.
